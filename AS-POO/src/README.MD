# Aplicação CRUD com Spring Boot

Este projeto foi desenvolvido usando Spring Boot e implementa operações CRUD (Create, Read, Update, Delete) para as entidades `Cliente` e `Carro`. A aplicação permite a interação com essas entidades através dos endpoints `/clientes` e `/carros`, utilizando um banco de dados para persistência.

## Autores do projeto

## Funcionalidades

A aplicação oferece as seguintes funcionalidades para as entidades `Cliente` e `Carro`:

### **Clientes**
- **Create**: Adicionar um novo cliente.
- **Read**: Listar todos os clientes ou buscar um cliente pelo ID.
- **Update**: Atualizar informações de um cliente.
- **Delete**: Remover um cliente pelo ID.

### **Carros**
- **Create**: Adicionar um novo carro.
- **Read**: Listar todos os carros ou buscar um carro pelo ID.
- **Update**: Atualizar informações de um carro.
- **Delete**: Remover um carro pelo ID.

## Tecnologias Utilizadas

- **Spring Boot**: Framework Java para construção de aplicações web e microserviços.
- **Spring Data JPA**: Para integração com o banco de dados e operações CRUD.
- **Banco de Dados**: Você pode escolher o banco de dados de sua preferência (exemplo: PostgreSQL, MySQL, H2, etc.).
